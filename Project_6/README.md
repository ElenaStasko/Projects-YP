# Анализ убытков приложения


## Описание проекта

Последние несколько месяцев компания терпит убытки, несмотря на огромные вложения в рекламу. Задача — разобраться в причинах и помочь компании выйти в плюс.


## Навыки и инструменты

- **python**
- **pandas**
- **matplotlib**
- **когортный анализ**
- **юнит экономика**
- **продуктовые метрики**

## Общий вывод
